Project B

Westinghouse JSON Page

A content-driven page that loads staff details and page sections from JSON, making it easier to update the site without editing the HTML manually.

Project details

  • Year completed: 2024
  • Role: Developer — created the content loading logic and page layout.
  • Tools: HTML, CSS, JavaScript, JSON.
  • Live URL: westinghouse staff page
  • GitHub: github.com/Jesusc315/Westinghouse-Website

What this project does

This page reads data from JSON and displays it as staff profiles, section headings, and supporting content. The goal was to make the page easier to maintain and faster to update.

Role and challenges

I built the JSON parsing and display logic, then matched the content structure to the visual layout. The challenge was ensuring cards and sections looked consistent as new data changed.
