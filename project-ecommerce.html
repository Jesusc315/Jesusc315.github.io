E-commerce Site

A polished storefront with product browsing, cart behavior, and checkout summary designed to show a complete shopping experience in one project.

Project details

  • Year completed: 2025
  • Role: Developer — built product pages, cart logic, and responsive layout.
  • Tools: HTML, CSS, JavaScript, responsive design.
  • Live URL: e-commerce-project
  • GitHub: github.com/Jesusc315

What this project does

This site lets visitors browse products, add items to a cart, and review a checkout summary. It demonstrates full e-commerce workflow thinking and user-friendly interface design.

Role and challenges

I developed the product display and cart logic and worked to keep the interface smooth on phones and desktops. The hardest part was organizing the layout so it felt polished while still being easy to use.
