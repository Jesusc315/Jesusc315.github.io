Project A

Virtual Store Shopping

A shopping experience with simple product selection, cart, and checkout interaction designed to feel polished on desktop and mobile.

Project details

  • Year completed: 2025
  • Role: Lead developer — designed UI, built cart logic, and handled user validation.
  • Tools: HTML, CSS, JavaScript, DOM manipulation.
  • Live URL: virtual-store-shopping-1e1y.vercel.app
  • GitHub: github.com/Jesusc315/Virtual-Store-Shopping

What this project does

This site lets users browse products, add items to a cart, and see live totals as they shop. It demonstrates handling user input, displaying updated information, and making the shopping flow easy to follow.

Role and challenges

I built the interaction logic and made sure the cart behaved correctly when quantities changed. The biggest challenge was validating inputs and preserving state while keeping the interface clean.
